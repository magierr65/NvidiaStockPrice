# Stock Price Prediction

Simple Python project for experimenting with stock price prediction using historical market data and machine learning models.

## Tech stack
- Python 3.12
- pandas, numpy
- scikit-learn
- matplotlib, seaborn
- alpaca-py
- uv (dependency management)

## Setup
bash
    `uv sync`

# Environment variables
## Create a .env file with:

ALPACA_API_KEY=your_key

ALPACA_API_SECRET=your_secret

## Run
`python src/nvidia_prediction.py`
